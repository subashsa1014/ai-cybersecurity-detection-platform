# Demo Video Scripts

Ready-to-read scripts for recording demo videos of the AI Cybersecurity Detection Platform.

## Full Demo Video (~90 seconds)

Ideal for: GitHub README, YouTube, portfolio website

### [0:00–0:08] Hook
> "Cyber attacks are growing every day — phishing emails, malicious files, and hidden malware. Manually analyzing each threat is slow, and traditional tools miss a lot."

**Screen:** Black screen or simple title slide. Voiceover only.

---

### [0:08–0:18] Intro
> "This is the AI Cybersecurity Detection Platform — an end-to-end system that uses machine learning to detect phishing URLs, scan suspicious files, and identify malware threats in seconds."

**Screen:** GitHub repo homepage. Show project title and description.

---

### [0:18–0:40] Feature 1: URL Phishing Detector
> "Let me show you how it works. First, I paste a suspicious URL here. The backend uses an XGBoost-based ML model that analyzes the URL's domain, length, HTTPS status, and suspicious patterns."

**Screen:** Frontend URL scanner page.

**Action:** Paste a test URL → Click Scan → Show risk score (0-100%) + explanation.

> "Within seconds, it returns a risk score, threat level, and an explainable AI breakdown of why the URL was flagged."

---

### [0:40–1:00] Feature 2: File Scanner + Malware Detection
> "Next, I upload a file — .exe, .pdf, .zip, anything. The platform runs both signature-based detection and AI-powered malware classification using Random Forest and XGBoost models."

**Screen:** Frontend file scanner page.

**Action:** Upload a test file → Click Scan → Show Safe/Suspicious/Dangerous indicator.

> "It instantly classifies the file as safe, suspicious, or dangerous — with visual threat indicators."

---

### [1:00–1:15] Feature 3: Dashboard + Authentication
> "All scans are logged here in the Threat Intelligence Dashboard. Users can authenticate securely with JWT, track their scan history, and monitor alerts with color-coded risk levels."

**Screen:** Dashboard page.

**Action:** Show scan history, alerts, threat statistics.

> "The entire platform is built with FastAPI on the backend, React on the frontend, MongoDB for data, and Docker for easy deployment."

---

### [1:15–1:25] Tech Stack (Quick)
> "Tech stack: FastAPI, React, Tailwind CSS, Scikit-learn, XGBoost, MongoDB, and JWT authentication — fully containerized with Docker."

**Screen:** Tech Stack section from README or brief code flash.

---

### [1:25–1:30] Call to Action
> "Check out the full source code on GitHub. Star the repo, fork it, and start building your own cybersecurity tools."

**Screen:** GitHub repo URL on screen.

## Recording Tips

1. **Use Loom** (free) — browser extension for screen + mic recording.
2. **Practice once silently** — click through the flow without speaking first.
3. **Clean your screen** — close other tabs, use dark mode, full-screen browser.
4. **Read naturally** — know the key points, don't memorize word-for-word.
5. **Good lighting** — face a window or use a desk lamp for clear audio.
6. **Background music** — add subtle background music in editing (optional).
